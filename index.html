    Edit First Default Image

    You want to do this just in case the JavaScript is disabled. If you don't, the first image may be incorrect if you changed the image's path or if you decided to change it within the JavaScript and didn't update it in the CSS.

  1. Open style.css file
  2. Find line 89, or #fullGallery
  3. Change the 'background-image' property to your default image path
  4. Open app.js
  5. Find line 8 and input the same image path as you put in the CSS
